A Catalog of Common HTML Tags


Common Text Tags

<p> Paragraph Tag

<b> Bold Tag (equivalent to <strong> Strong Tag)
<i> Italic Tag (equivalent to <em> Emphasis Tag)

(The reason why you would use Strong and Emphasis tags over bold and italic ones is if you want to include certain text into the page's metadata, usually for affecting how it's detected by search engines. Is also utilized by screen readers.)

<strike> Strikethrough Tag

<br> Line Break Tag (Inserts a line break)

User Input Examples

<button> Button Tag
(Used for basic user input)

<input> Input Tag
(Defaults to single-line user text input. The type property determines the kind of input element it is)

<textarea> Textarea Tag
(Used for multi-line user text input. Default size can be adjusted using rows and cols properties.)

<input type="range"> Range Input Tag
(Creates a slider which can have its value adjusted by the user, minimum and maximum values can be set in the element's properties via its HTML tag or through Javascript)
