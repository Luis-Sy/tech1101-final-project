Commands to Retreive Elements From the Page

document.GetElementById("insert id here")
(Searches for the first element with the specified ID in the parentheses, returns NULL if nothing is found.)


document.GetElementByClass("insert class here")
(Searches for the first element with the specified class in the parentheses, returns NULL if nothing is found.)


Common Commands

innerHTML()
Returns all text and tags contained in an element


innerText()
Only returns all text contained in an element


appendChild(child)
Adds the element passed into the function as a child to another element


removeChild(child)
Removes the first child of an element by default, if a child is passed in the function, remove the first instance found


createElement("tag")
Creates a new element of the tag specified in the function
